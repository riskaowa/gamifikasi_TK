# 🎮 Gamifikasi PAUD  
✨ Platform Gamifikasi Pembelajaran Anak Usia Dini Berbasis Web

Gamifikasi PAUD adalah aplikasi web yang dirancang untuk **mendukung proses belajar anak PAUD** dengan pendekatan **bermain sambil belajar**.  
Sistem ini menggunakan **poin, level, dan reward** untuk meningkatkan motivasi, keterlibatan, dan semangat belajar anak.

---

## 🌈 Fitur Utama
- 👶 **Manajemen Anak PAUD**
- ⭐ **Sistem XP (Experience Point)**
- 🆙 **Level Up Otomatis**
- 🏆 **Leaderboard Sederhana**
- 🎖️ **Reward & Achievement**
- 📊 **Monitoring Perkembangan Anak**
- 🎨 **Desain Ramah Anak**

---

## 🧠 Konsep Gamifikasi
Gamifikasi diterapkan dengan pendekatan:
- Aktivitas belajar → **XP**
- Akumulasi XP → **Level**
- Pencapaian tertentu → **Badge / Reward**
- Kompetisi sehat → **Leaderboard**

Pendekatan ini membantu anak:
- Lebih termotivasi
- Belajar dengan perasaan senang
- Mengembangkan konsistensi & kepercayaan diri

## ⚙️ Instalasi & Menjalankan Project

```bash
